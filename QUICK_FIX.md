# Быстрое решение проблем

## Шаг 1: Откройте консоль Chrome

1. Запустите приложение: `flutter run -d chrome`
2. Нажмите **F12** (или Cmd+Option+I на Mac)
3. Откройте вкладку **Console**

## Шаг 2: Проверьте ошибки

Отправьте сообщение и посмотрите на ошибки в консоли.

### Если видите ошибку CORS:
```
Access to XMLHttpRequest blocked by CORS policy
```
**Решение:** Убедитесь, что бэкенд запущен и CORS настроен правильно.

### Если видите ошибку 500:
```
Failed to get response: 500
```
**Решение:** 
1. Проверьте, что файл `backend/.env` существует
2. Убедитесь, что в нем есть ваш DeepSeek API ключ:
```
DEEPSEEK_API_KEY=sk-your-key-here
PORT=3000
```
3. Перезапустите бэкенд: `cd backend && npm start`

### Если видите "Connection refused":
**Решение:** Запустите бэкенд:
```bash
cd backend
npm install  # если еще не установлено
npm start
```

## Шаг 3: Проверьте Network tab

1. В DevTools откройте вкладку **Network**
2. Отправьте сообщение
3. Найдите запрос к `/api/chat`
4. Посмотрите:
   - **Status** - должен быть 200
   - **Response** - должен содержать JSON с ответом

## Быстрая проверка бэкенда

```bash
# Проверка здоровья сервера
curl http://localhost:3000/health

# Должно вернуть: {"status":"ok"}
```

## Если ничего не помогает

Запустите скрипт проверки:
```bash
./check_setup.sh
```

Он покажет, что именно не настроено.
